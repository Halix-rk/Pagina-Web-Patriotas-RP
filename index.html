Licencias del Servidor

Estas son las licencias disponibles dentro del servidor. Cada una representa un rol que los usuarios pueden obtener según las normas del servidor.

  • Licencia Clase A1: Autoriza conducir motocicletas de baja cilindrada y scooters pequeños.
  • Licencia Clase A2: Permite manejar motocicletas de mayor potencia y motos urbanas.
  • Licencia Clase A3: Habilita para conducir triciclos y motos con carro lateral.
  • Licencia Clase A4: Permite el uso de motocicletas deportivas o de alta cilindrada.
  • Licencia Clase A5: Autorización para motos profesionales o de uso especializado.
  • Licencia Clase B: Licencia básica para conducir autos particulares.
  • Licencia Clase C: Permite conducir vehículos de carga liviana.
  • Licencia Clase D: Autoriza conducir maquinaria pesada o vehículos especiales.
  • Licencia Clase E: Habilita el manejo de vehículos de transporte público o taxis.
  • Licencia Clase F: Permite conducir vehículos de emergencia (ambulancias, carros de bomberos, etc.).
  • Licencia de Armas: Rol especial para usuarios autorizados a portar armas dentro del rol.
